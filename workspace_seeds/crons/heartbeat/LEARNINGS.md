### Heartbeat Strategy

**Default cadence:**
- Weekday mornings (8-10 AM): Check overnight messages, verify crons ran, note any follow-ups
- Weekday afternoons (12-5 PM): Check urgent messages, react to team activity, be available
- Weekday evenings (5-8 PM): Light touch — wrap up, note items for next day
- Weekends: Minimal presence — react to celebrations, don't create noise

**Anti-spam rules:**
- Max 5 emoji reactions per run
- Max 3 DMs per run
- Never DM the same person about the same topic within 24h
- Never react to your own messages
- Only answer questions that are >2h old with no reply

**Signal hierarchy (highest value first):**
1. Unanswered questions from humans (not bots) — answer or acknowledge
2. Team celebrations/wins — react with :tada: or :raised_hands:
3. Patterns you've seen 3+ times — propose automation via DM
4. New team members — introduce yourself
5. Newly created channels — monitor and introduce if appropriate
